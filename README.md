# 📱 JobsWork Mobile App – Recruitment Platform (React Native)

A mobile application for the JobsWork recruitment system, built with React Native, allowing job seekers and employers to access recruitment services anytime and anywhere.

## 📌 Features

- User registration and login
- Secure authentication via RESTful API
- Role-based access (Employer – Candidate)
- Job search and filtering
- Job application management
- Profile and CV management
- Job notification system
- Saved jobs and application history
- Responsive UI for Android & iOS
- API integration with Laravel backend

## 🛠️ Technologies Used

- Mobile Framework: React Native
- Language: JavaScript
- Backend API: Laravel RESTful API
- Database: MySQL (via API)
- Tools: Git, GitHub, VS Code, Android Studio, Postman
